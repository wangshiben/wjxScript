# 问卷星刷问卷

> 此代码需要运行环境: selenium库以及requests库  还有chromeDiver的设置

## 如何设置答题内容:

可以参考demo.Py，如果出现代码运行时找不到点哪个题目，请自行修改mainUtil.Py中的逻辑或新增

> 注意:问卷星对于每个IP段有限制，如果短时间内大量答题会引起人机验证，如果是100份答题以上的，建议
> 重写Runner.Py中的new_ip函数(获取新的代理IP)
